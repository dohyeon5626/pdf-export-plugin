This Figma plugin lets you export selected frames as a single merged PDF file. You can drag and drop to reorder frames before exporting, so the pages appear in exactly the order you want. Simply select the frames, components, or instances you need, rearrange them with drag-and-drop, and click "Export PDF" to download. No extra steps or external tools required.
